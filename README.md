# BMS-BookManagerSystem
Javaweb写的仓库管理系统，使用jsp+jdbc+JavaScript+css，还用了部分Ajax技术，能力有限，真的在学了😫😫😫
主要页面都是用jsp语言写的，使用tomcat8.5和mysql8.0.32，修改数据库登录用户名密码在util.DButil.java
